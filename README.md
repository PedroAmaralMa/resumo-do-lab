# resumo-do-lab
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na Fatec

# Aula 01 Introdução ao Azure:

Nesta aula tivemos uma introdução ao Microsoft Azure, explorando sua interface e aprendendo a configurar idioma e aparência da ferramenta. Também navegamos pela página inicial, pelo menu lateral com todos os recursos disponíveis e entendemos onde acessar a documentação oficial e a seção Ajuda + Suporte, incluindo a visualização de informações relacionadas à assinatura e seus diferentes níveis de suporte.
Além disso, conhecemos alguns serviços disponíveis na plataforma, como o Azure Bastion e o Azure Virtual Desktop, compreendendo onde eles se localizam no portal e como fazem parte do conjunto de recursos oferecidos pelo Azure.

# Aula 02 Criando uma máquina virtual: 

Nesta aula aprendemos a criar e configurar uma máquina virtual no Microsoft Azure, passando por todas as etapas de configuração. Vimos como escolher a região do datacenter (e como isso impacta no preço e na latência), criar o grupo de recursos, definir usuário e senha de administrador, selecionar o sistema operacional e habilitar o acesso via RDP para posterior conexão com a máquina. Também exploramos a seção de rede, entendendo configurações como rede virtual, sub-rede, IP público, regras de entrada e as opções de balanceamento de carga.

Na parte de rede, aprendemos sobre recursos como o Azure Load Balancer, compreendendo sua função no balanceamento de tráfego entre múltiplas máquinas virtuais para garantir alta disponibilidade e melhor distribuição de carga. Além disso, vimos a importância do monitoramento, configurando alertas e definindo o desligamento automático da máquina virtual com ajuste de fuso horário, evitando desperdício de recursos caso a máquina permaneça ligada sem necessidade.

Por fim, realizamos a revisão final das configurações antes da criação da máquina virtual, analisando possíveis mensagens de erro e recomendações apresentadas pelo próprio Azure, entendendo como essas validações ajudam a garantir uma implantação adequada do recurso.
